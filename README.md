# Time::Utils

`Time::Utils` extracts some of ActiveSupports's time utilities.

## Installation

Add this line to your application's Gemfile:

    gem 'time-utils'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install time-utils
